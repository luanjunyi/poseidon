- (void) fromJSON: (NSDictionary*)  data

This method will take an NSDictionary (which may have nested NSArray or NSDictionary values) that can represent a JSON structure, to populate all its field values. At the beginning it will clear all its current field values. This method is useful for the services that use SBJSON to parse the REST API returned data to NSArray/NSDictionary representing values.

Parameters:
dataJSON structure to parse.

- (NSDictionary *) toJSON

This method is a reverse operation of the fromJSON. It is useful for the services that use SBJON to write NSArray/NSDictionary values to JSON String to be posted to REST API.
